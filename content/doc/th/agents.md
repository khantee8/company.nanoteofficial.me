# เอเจนต์ทั้ง 6

แต่ละแผนกคือเอเจนต์ Claude อัตโนมัติที่มีหน้าที่ประจำ และยัง **ทำงานร่วมกัน** ด้วย — เอเจนต์ที่รันทีหลังในวันเดียวกันจะเห็นผลงานก่อนหน้าของเพื่อนร่วมงานและต่อยอดได้

## Finance
**ผู้ช่วยวิเคราะห์กองทุนรวมไทย** แต่ละรอบเปรียบเทียบกองทุนจริงตามธีมหมุนเวียน (กองดัชนีสหรัฐ, เทคโนโลยีโลก/เซมิคอนดักเตอร์, หรือกองลดหย่อนภาษี SSF/RMF/Thai ESG) พร้อมอ้างอิงทุกค่าธรรมเนียม AUM และผลตอบแทน

## CyberX
**สรุปภัยคุกคามรายวัน** ดึง CVE ที่เพิ่งถูกโจมตีจากแคตตาล็อก CISA KEV (`api`) และเพิ่มคำแนะนำที่ค้นจากเว็บ (`web · cited`) พร้อมระดับความรุนแรงและวิธีรับมือ

## AI R&D
**เรดาร์เทคโนโลยี** ติดตาม repository ที่กำลังมาแรง และค้นคว้า paper, release และเครื่องมือจริงในโฟกัสหมุนเวียน (AI agents หรือ LLM infrastructure)

## Marketing & Social Media
**แผนคอนเทนต์ที่ขับเคลื่อนด้วยดีมานด์** อ่านดีมานด์จริงจากนักพัฒนา (Hacker News, Dev.to, analytics ของเว็บ) ค้นคว้าสัญญาณ และเสนอคอนเทนต์ที่ผูกกับเทรนด์สด

## Operations
**สกอร์การ์ดสุขภาพการดีพลอย** อ่านสถานะ CI/CD จาก Vercel + GitHub และชี้สิ่งสำคัญที่สุดที่ควรแก้วันนี้

## CEO
