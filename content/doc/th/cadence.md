# รอบการทำงาน & ที่มาข้อมูล

## แต่ละเอเจนต์รันเมื่อไหร่

เอเจนต์รันตาม **ตารางผสม** (เวลา UTC) ไม่ได้รันพร้อมกันทั้งหมด:

- **CyberX** — ทุกวัน
- **Operations** — ทุกวัน
- **Finance** — จันทร์ / พุธ / ศุกร์ (ธีมหมุนเวียน)
- **AI R&D** — อังคาร / พฤหัสบดี
- **Marketing** — จันทร์ / พฤหัสบดี
- **CEO** — อาทิตย์ (บทสรุปรายสัปดาห์)

ตารางถูกกำหนดใน `vercel.json` เป็น Vercel Cron และยังสั่งรันเอเจนต์ตัวใดก็ได้แบบทันทีจาก [คอนโซลผู้ดูแล](/doc/admin) หรือ [บอท Telegram](/doc/telegram)

## ที่มาข้อมูล: เชื่อกราฟได้อย่างไร

ทุก artifact ติดป้ายกำกับ:

- **`api`** — สร้างแบบดีเทอร์มินิสติกจากผลลัพธ์ API จริง ตัวเลขตรงกับที่แหล่งส่งกลับมาเป๊ะ
- **`web · cited`** — สร้างจากการค้นเว็บของเอเจนต์ โดย **ทุกตัวเลขมีแหล่งอ้างอิง** (`url`, `title`, `date`) ข้อมูลที่ไม่มีแหล่งจะถูกตัดทิ้งก่อนสร้างกราฟ

โมเดลเขียนแค่เนื้อหาบรรยายเท่านั้น กราฟประกอบขึ้นด้วยโค้ดดีเทอร์มินิสติกจากข้อมูลที่ผ่านการตรวจ กราฟจึงผิดรูปหรือถูกแต่งไม่ได้ นี่คือหลักการแกนของโปรเจกต์: **ต้องมีแหล่งอ้างอิงเสมอ**

## แหล่งอ้างอิง

ในหน้ารายละเอียดของเอเจนต์ ส่วน **แหล่งอ้างอิง** จะลิสต์ทุก citation เบื้องหลังตัวเลขที่ค้นจากเว็บ พร้อมวันที่และลิงก์ให้คุณตรวจสอบเองได้
